Welcome to Chameleon!
===================


This extension allow to run other extensions from **Opera** add-ons

Feedback [chameleon.ext@gmail.com](mailto:chameleon.ext@gmail.com)

##### Supported items

 - browser_action
 - background
 - content_scripts (run at document\_end only)
